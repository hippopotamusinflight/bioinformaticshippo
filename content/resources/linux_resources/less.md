## search

```bash
n for next

```

## go to
```
100g     go to the 100th line (or `less +100g bigfile.txt`)
50p      go to 50% into the file.
100P     go to the line containing 100th byte.

^f       page down
^b       page up
G        go to the end of buffer
gg       go to the beginning of the buffer
```
